# Assets - Carpeta de Recursos

## Archivos Aquí

Esta carpeta contiene todos los recursos estáticos de la aplicación ECUAYAPA.

### logo.png

Tu logo debe ser colocado aquí como `logo.png`. 

**Recomendaciones:**
- Formato: PNG con fondo transparente
- Resolución: 512x512 px o superior
- Tamaño del archivo: Menos de 500KB
- Estilo: Cuadrado o circular para mejor presentación

El logo se muestra en el banner de forma épica con:
- Anillo rotatorio animado
- Efecto glow pulsante
- Sombra dramática
- Animación flotante suave
- Responsivo (responsive) para todas las pantallas

## Estructura Responsive

- **Mobile**: El logo se muestra centrado arriba del contenido
- **Desktop**: El logo aparece a la derecha del texto con tarjetas informativas
